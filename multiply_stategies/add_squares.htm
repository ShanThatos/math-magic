Adding Squares In The Form: a2 + b2 - (a-b)2:

A.  From algebra we learn:

a2 + b2 - (a-b)2 = 2ab

B.  Basically just multiply a and b together then double it.

C.  Examples:

Ex [1]  92 + 32 - 62 = ________

a.  The answer is 2(9)(3) = 54.

Ex [2]  (25 + 8)2 + 252 - 82 = ________

a.  In this example a=33, since 25 + 8 = 33.

b.  The answer is 2(33)(25) = 50 x 33 = 1650.  See Multiplying By 50.
