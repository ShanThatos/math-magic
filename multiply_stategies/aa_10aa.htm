(**) Adding Squared Numbers In The Form: a2 + (10a)2:

   A.  This method is very easy since from algebra we know:

a2 + (10a)2 = 101a2

   B.  Therefore, you square the smaller number, then multiply by 101 for the answer.

Ex [1]  142 + 1402 =_________.

a)      142 = 196.

b)      196 x 101 = 19796.

c)      The answer is 19796.

Ex [2]  312 + 3102 =_________.

a)      312 = 961.  See Squaring Numbers.

b)      961 x 101 = 97061.

c)      The answer is 97061.
