Multiplying By 101:

1. When multiplying a 2 digit number by 101, simply write the number down twice.

Ex [1]  53 x 101 = 5353.

Ex [2]  49 x 101 = 4949.

2.  Sometimes there are variations of 101 like 201, 203, 104, etc.  In problems like these you can use the following:

Ex [1]   23 x 203 =_________.

a)      Multiply 23 x 2 = 46.

b)      Multiply 23 x 3 = 69.

c)      The answer is 4669.

3.  When multiplying a 3 digit number by 101 the following rules apply:

a. Write down the last 2 digits of the number.

b. Take the number in the hundred’s digit and add back to the number.

Ex [1]  453 x 101 =_________.

a)      Write 53.

b)      4 + 453 = 457.  Write 457

c)      The answer is 45753.

Ex [2]  998 x 101 =_________.

a)      Write 98.

b)      9 + 998 = 1007.  Write 1007.

c)  The answer is 100798.
