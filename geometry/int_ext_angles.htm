Ex [2]  If the sum of the interior angles of a regular polygon is 1440o, then the number of sides of the polygon is _____.

a.  Since the sum of the interior angles is 180 (n-2), we can set up the equation 180 (n-2) = 1440.

b.  1440/180 = 8.  So n - 2 = 8, or n = 10.

c.  The answer is 10.
