Multiplying Mixed Numbers Whose Fractions Are The Same

A.  From algebra we learn:                       

a b  x  d b  =  ad + b (a + d) + (b/c)2

    c        c               c                      

     B.     Use the following rules:

1.      Multiply the two whole numbers together.

2.      Add the two whole numbers together and multiply by the fraction.

3.      Add step 1 and step 2 for the whole number to the answer.

4.      Square the fraction.  This is the fraction for the answer.

          Ex [1]  5 1 x  3 1  =

                                     4              4                           (mixed number).

a)      5 x 3 = 15.

b)      (5 + 3) x 1/4 = 8 x 1/4 = 2.

c)      15 + 2 = 17.  Write 17.

d)      (1/4)2 = 1/16.  Write 1/16.

e)      The answer is 17 1/16.

                                       Ex [2]  12 3 x 4 3  =

                                          8           8                         (mixed number).

 a)      12 x 4 = 48.

b)      (12 + 4) x 3/8 = 16 x 3/8 = 6.

c)      48 + 6 = 54.  Write 54.

d)      (3/8)2 = 9/64.  Write 9/64.

e)      The answer is 54 9/64.
